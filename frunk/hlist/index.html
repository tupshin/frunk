Module that holds HList data structures and implementations

Typically, you would want to use the hlist! macro to make it easier for you to use HList.

let h = hlist![1, "hi"];
assert_eq!(h.length(), 2);
let (a, b) = h.into_tuple2();
assert_eq!(a, 1);
assert_eq!(b, "hi");Run
